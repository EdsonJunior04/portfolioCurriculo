Sobre o Site

Desenvolvimento do Projeto

Este projeto foi desenvolvido como um portfólio online para apresentar as habilidades adquiridas ao longo do meu aprendizado em desenvolvimento web. O site inclui funcionalidades para cadastro de currículos, listagem de postagens e uma interface intuitiva para os usuários interagirem com as informações.

Ferramentas Utilizadas

  • Angular: Para o desenvolvimento da interface do usuário, proporcionando uma experiência rica e responsiva.
  • Java: Utilizado para construir a API que gerencia a lógica do servidor e as requisições de dados.
  • MySQL: Banco de dados relacional utilizado para armazenar informações dos usuários e currículos.
  • GitHub Actions: Implementado para automação de integração contínua e entrega contínua (CI/CD).
  • Docker: Utilizado para containerização das aplicações, garantindo consistência em diferentes ambientes.
  • Kubernetes: Gerenciador de containers para orquestrar a aplicação em um cluster.
  • Terraform: Ferramenta de infraestrutura como código (IaC) para provisionamento dos recursos na AWS.
  • Ansible: Utilizado para automação de configuração e gerenciamento de servidores.

Implantação na AWS Academy

O site foi implantado na AWS Academy, utilizando uma esteira de automação com GitHub Actions. O processo de implantação consiste nas seguintes etapas:

  1. Build: A aplicação é construída e testada automaticamente usando GitHub Actions.
  2. Containerização: A aplicação é empacotada em um container Docker, garantindo que funcione da mesma forma em qualquer ambiente.
  3. Orquestração: Os containers são gerenciados por Kubernetes, permitindo escalabilidade e resiliência.
  4. Provisionamento: Recursos da AWS são provisionados usando Terraform, permitindo fácil gerenciamento e replicação da infraestrutura.
  5. Configuração: Ansible é utilizado para automatizar a configuração dos servidores, garantindo que todas as dependências estejam instaladas e configuradas corretamente.

Com essa abordagem, consegui criar um fluxo de desenvolvimento eficiente, garantindo que a aplicação esteja sempre atualizada e disponível para os usuários.
